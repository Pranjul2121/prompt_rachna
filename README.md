# Prompt Rachna

An interactive storytelling web experience built using **HTML5 Canvas**, **GSAP animations**, and smooth scroll interactions.
The project visually explains how a prompt evolves into a structured idea through animated illustrations and storytelling.

## ✨ Features

* Interactive storytelling flow
* Smooth animations using GSAP
* Custom canvas illustrations
* Responsive layout
* Lightweight and fast loading

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* HTML Canvas
* GSAP (GreenSock Animation Platform)

## 🚀 Live Demo

Deployed using **Vercel**.
